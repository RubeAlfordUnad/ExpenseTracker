# Nexora

A personal finance iOS app built with SwiftUI.

## Features
- Authentication
- Expense tracking
- Monthly budget dashboard
- Credit card debt tracking
- Recurring monthly payments
- Statistics and charts

## Tech Stack
- SwiftUI
- UserDefaults
- Charts
- MVVM-style structure
